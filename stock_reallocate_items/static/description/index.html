Overview

     Most of company suffer about order management and will lose opportunity in this point? So, our module is helping to make this paint point to get more effective. In this function is working based on sale order significant. This function can commutate several products in each order easily. In addition, also have additional option to setting about approved process.

     In this module will arrange the module to appear on inventory module, this module will sort in operation
Reallocate.

The system will show function on display and default operation type in on "Delivery order". Then user can choose in source location for search about product to allocation.

Then user can input in product name for allocation. The operation will appear transfer order refer on sale order.

For next step, user can put the number of commit product for change several reserved.

However, there are some conditions in this module will get more efficient using. Consist of

  1. A number of commits must less than available quantity.
  2. A number of commits must less than demand quantity.

If there are some conditions has working, the system will show alert popup.

When status was changed to "Done", in the transfer order will update status or log note in odoo system according to reallocate process.

In addition, our function has optional about approval process. It can add user who has access to approved process before done process.
